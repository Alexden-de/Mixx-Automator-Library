# HTTP Download → CSV-Mapping

**Kategorie:** Beispiel | **Aktionen:** 2

## Was der Job macht

1. **HTTP Download** — laedt eine CSV-Datei von einer URL
2. **Daten-Mapping** — mappt Felder (Spalten umbenennen, Werte transformieren)

## Anpassen nach dem Import

- **HTTP Download → URL**: deine eigene CSV-Quelle eintragen
- **DataMapping → MappingRules**: Zielspalten anpassen
- **Zeitplan aktivieren** wenn regelmaessiger Abruf gewuenscht

## Verwendete Platzhalter

- `{date}` im Dateinamen → `2026-04-17`
- `{datetime}` → `2026-04-17_10-30-15`
